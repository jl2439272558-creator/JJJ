# Warm WorkLog Desktop (暖色便签)

一款极简风格的桌面便签与效率工具，集成了待办事项管理、专注番茄钟与操作日志功能。采用清爽的磨砂玻璃 UI 设计，支持桌面边缘吸附与自动隐藏，助您保持专注与高效。

## ✨ 核心功能

### 📝 极简便签 (Sticky Notes)
- **快速记录**：顶部常驻输入框，输入内容后回车即刻创建待办。
- **任务管理**：支持标记完成、删除（物理删除）及长按标记为“紧急”（红色高亮）。
- **智能排序**：未完成任务优先展示，已完成任务自动沉底。
- **无干扰设计**：隐藏式滚动条，界面清爽整洁。

### 🍅 专注番茄钟 (Focus Timer)
- **可视化进度**：环形进度条直观展示剩余时间。
- **灵活设置**：支持 1-120 分钟的自定义专注时长调节。
- **沉浸体验**：计时结束时弹出优雅的磨砂玻璃风格通知。

### 📋 操作日志 (Activity Logs)
- **历史回溯**：自动记录最近 7 天的便签操作记录（新建、完成、删除等）。
- **相对日期**：采用“今天”、“-1”、“-2”等相对日期展示，直观清晰。
- **隐私管理**：支持一键清空所有历史日志。

### 🖥️ 桌面体验 (Desktop Experience)
- **边缘吸附**：拖动窗口至屏幕边缘（上/左/右）即可自动吸附。
- **自动隐藏**：吸附后自动收起为半透明气泡图标，鼠标悬停即可展开。
- **单例运行**：防止程序多开，重复打开时自动唤醒已有窗口。
- **双色主题**：内置亮色与暗色两套磨砂玻璃主题。

## 🛠️ 技术栈

- **GUI 框架**: PySide6 (Qt for Python 6)
- **数据库**: SQLite + SQLAlchemy (本地存储，数据安全)
- **打包工具**: PyInstaller (单文件绿色版)


### 源码运行
1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```

2. 启动应用:
   ```bash
   python worklog_desktop/app/main.py
   ```
